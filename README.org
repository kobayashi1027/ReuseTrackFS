* What is this?
  reuse_tracker is a tool to collect reuse log using FUSE
* Setup
  1) Install submodule
     #+BEGIN_SRC sh
     $ git submodule init
     $ git submodule update
     #+END_SRC

  2) Create mount point
     #+BEGIN_SRC sh
     # example
     $ mkdir /mnt/home
     #+END_SRC

* Usage
  #+BEGIN_SRC sh
  # mount your directory to mount point
  $ python main.py /your/directory /mnt/home
  #+END_SRC
